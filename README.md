# TP Express.js + PostgreSQL + EJS

## 📌 Description
Application web simple de gestion des utilisateurs développée avec :
- Node.js / Express
- PostgreSQL
- EJS (templates)

Permet de :
- Ajouter un utilisateur
- Modifier un utilisateur
- Supprimer un utilisateur
- Afficher la liste des utilisateurs
